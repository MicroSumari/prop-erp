# Property Management ERP System - Frontend

React-based frontend for the Property Management System integrated with ERP.

## Installation

```bash
npm install
```

## Configuration

Create a `.env` file in the frontend directory:

```
REACT_APP_API_URL=http://localhost:8000/api
```

## Development

```bash
npm start
```

The app will run on `http://localhost:3000`

## Build

```bash
npm build
```

## Features

- Property Management
- Tenant Management
- Lease Tracking
- Rent Collection
- Maintenance Tracking
- Expense Management
- Dashboard with Analytics
